# My Dictionary
## Basic Functionality
The base idea for this project is **tracking new words learned in a foreign language**.I am developing it not only to learn the process of development and to add projects to my portfolio, but also because I genuenly need such a program. 

Currently, I am learning german to do an Ausbildung als Fachinformatiker Anwendungsentwicklung (Programmer for App Development). It's basically a 3 year learning program to acquire a profession, with no previous experience needed.

## It does 4 Main things:

* shows them in a table
* saves them in local storage
* lets the user add/delete entries
